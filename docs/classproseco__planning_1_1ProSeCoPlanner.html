◆ isTerminal()

bool proseco_planning::ProSeCoPlanner::isTerminal ( ) const

Evaluates whether the planner has reached a terminal state.

Returns
true If the environment reached a collision or invalid state, as well as when the maximum number of steps or maximum duration has been reached. But also, when the desire is fulfilled or the terminal condition of the scenario is met.
false Otherwise.

◆ overrideActionsPredefined()

void proseco_planning::ProSeCoPlanner::overrideActionsPredefined ( ActionSet &  actionSet) const

Overrides the actions chosen by predefined agents.

If an agent is predefined an action will still be chosen based on the final selection policy, so in order to make sure that it actually executes a predefined action it is overwritten here.

Parameters
actionSetThe action set that is to be executed.
